## вҡЎ Quick Start (TL;DR)

### Requirements
- Node.js v18+
- npm v9+
- Credentials from MongoDB, Firebase, Gmail

### Steps
1. Get credentials (15 min)
2. Fill .env files (5 min)
3. Run: `start-dev.bat` (Windows) or `./start-dev.sh` (Mac/Linux)
4. Open: http://localhost:5173

### URLs
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
